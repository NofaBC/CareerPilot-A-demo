# CareerPilot AI™ Demo

This repo contains the **single-file UI demo** for CareerPilot AI™ – an AI-powered job search assistant that helps users:

- Turn their job profile into tailored applications  
- Find high-fit roles (not random listings)  
- Prepare for interviews with targeted questions and talking points  

## What This Repo Is

- A **static HTML demo** (`index.html`)
- Used to showcase the look and feel of CareerPilot AI™
- Great for quick previews, portfolio, and design reference

There is **no backend, database, or real AI** wired up in this demo. All logic is front-end only.

## Live Demo

Deployed with Vercel:

- https://career-pilot-ai-psi.vercel.app/ *(or your updated demo URL)*
